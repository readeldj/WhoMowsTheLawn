# WhoMowsTheLawn
Map out lawn service pricing by end users.

Start of the project for end of class.

User can create Login and setup profile.
User adds:
  Address for mapping.
  Lawn Provider Website and/or Facebook page.
  Price for service.
  Comments.

No login required to search mapped data.


Bower Dependencies:
  skeleton: 2.0.4
  jquery: 2.1.3
  firebase: 2.2.3
